1 数据类型

1.1 Swift的数据类型分三类：结构体，类和枚举

每种数据类型都有：
属性 - 关联到类型的值
构造器 - 初始化类的实例的代码
实例方法 - 关联到类型的函数，可以被该类型的实例调用
类方法 - 也叫静态方法，关联到类型的函数，可以被类型调用

Swift下一些基本的数据类型都是结构体，比如：
数字：Int, Float, Double
布尔：Bool
文本：String, Character
集合：Array, Dictionary, Set

这种分类的依据是什么？他们之间有什么不同点？
